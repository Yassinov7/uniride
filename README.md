# UniRide 🚍

نظام ذكي لتنظيم نقل الطلاب بين الجامعة والسكن.  
يسمح للطلاب بتقديم طلبات حجز أسبوعية، وللمشرفين بإدارة الرحلات والباصات وتتبع حالة كل طالب بشكل مرن وعملي.

---

## 📌 الميزات

- ✅ تقديم طلبات الحجز من قبل الطالب لأسبوع كامل دفعة واحدة.
- 🚌 إدارة الباصات والرحلات (ذهاب وعودة) من قبل المشرف.
- 🧠 توزيع الطلاب على الباصات حسب الجنس والجامعة والمنطقة.
- ⏱️ تحديد توقيت الذهاب والعودة من قبل الطالب.
- 💰 نظام محفظة للخصم والإضافة، مع سجل مالي مفصل.
- 🔔 تنبيهات وإشعارات فورية (Toast).
- 🌙 دعم الوضع المستقل PWA (تثبيت التطبيق على الهاتف).

---

## ⚙️ التقنيات المستخدمة

- **Next.js 15+** – App Router + Client Components
- **Supabase** – قاعدة البيانات + Auth + RLS + Storage
- **Tailwind CSS** – تصميم متجاوب وسريع
- **react-hot-toast** – التنبيهات
- **sweetalert2** – نوافذ التأكيد
- **xlsx** – تصدير بيانات الرحلات
- **dayjs** – التعامل مع التواريخ
